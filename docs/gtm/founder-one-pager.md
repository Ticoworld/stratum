# Stratum for Founders and Operators

Stratum is watchlist intelligence for a finite list of target companies. It is built for corporate development and investor teams that need a clear record of what changed between ATS checks.

The workflow is simple:
- Add a company to a watchlist.
- Save a point-in-time brief from supported ATS sources.
- Run manual or scheduled refreshes.
- Compare the latest saved brief with the previous one.
- Review meaningful changes in the in-app inbox.

What it solves:
- Manual spot checks are easy to miss.
- ATS review is hard to compare over time.
- Teams need a repeatable record, not just a live query.

What makes it different:
- Narrow by design.
- Point-in-time saved artifacts.
- Explicit latest-vs-previous comparison.
- In-product inbox instead of an external alert system.
- Honest limits around source coverage and match quality.

What it does not claim:
- Full-company intelligence.
- ATS omniscience.
- External alerts.
- Generic AI-agent behavior.
