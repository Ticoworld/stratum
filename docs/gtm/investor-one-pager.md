# Stratum for Investors and Funders

Wedge: a narrow watchlist intelligence product for target companies.

Buyer pain: corp dev and investor teams miss hiring changes between spot checks, and manual ATS review is hard to compare over time.

Product mechanism: Stratum saves point-in-time briefs, compares latest vs previous, supports scheduled refresh, and surfaces meaningful changes in an in-app inbox.

Why the workflow matters: it turns noisy, one-off checks into a repeatable monitoring record that is easy to review and explain.

Why Stratum is narrow on purpose: the product is designed around a finite target list and supported ATS sources so the user gets honest, reviewable signals instead of broad but vague company-intelligence claims.

Intentionally outside scope:
- Full-company intelligence
- External alerting
- Broad market monitoring
- Claims that ATS alone explains the whole company
- Generic AI output without a saved brief and comparison record
