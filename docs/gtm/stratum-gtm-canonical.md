## 4. Founder-Facing One-Pager

Stratum is watchlist intelligence for a finite list of target companies. It is built for corporate development and investor teams that want a clear record of what changed between ATS checks.

The workflow is simple:
- Add a company to a watchlist.
- Save a point-in-time brief from supported ATS sources.
- Run manual or scheduled refreshes.
- Compare the latest saved brief with the previous one.
- Review meaningful changes in the in-app inbox.

What it solves:
- Manual spot checks are easy to miss.
- ATS review is hard to compare over time.
- Teams need a repeatable record, not just a live query.

What makes it different:
- Narrow by design.
- Point-in-time saved artifacts.
- Explicit latest-vs-previous comparison.
- In-product inbox instead of an external alert system.
- Honest limit language around coverage and match quality.

What it does not claim:
- Full-company intelligence.
- ATS omniscience.
- External alerts.
- Generic AI-agent behavior.

## 5. Investor / Funder One-Pager

Wedge: a narrow watchlist intelligence product for target companies.

Buyer pain: corp dev and investor teams miss hiring changes between spot checks, and manual ATS review is hard to compare over time.

Product mechanism: Stratum saves point-in-time briefs, compares latest vs previous, supports scheduled refresh, and surfaces meaningful changes in an in-app inbox.

Why the workflow matters: it turns noisy, one-off checks into a repeatable monitoring record that is easy to review and explain.

Intentionally outside scope:
- Full-company intelligence
- External alerting
- Broad market monitoring
- Claims that ATS alone explains the whole company
- Generic AI output without a saved brief and comparison record
